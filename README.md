Welcome to the BTC to CZK converter!

To run this locally:

1. npm install
2. Create .env where you need to have 2 values:
    a. NEXT_PUBLIC_TATUM_TESTNET_API_KEY
    b. NEXT_PUBLIC_API_BASE_URL

3. After adding those 2 values you can run: npm run dev
